# SailPoint Demo App

As seen at [//Build 2015](http://channel9.msdn.com/Events/Build/2015/3-680) presented by Erich Gamma, Chris Dias and John Papa.

## Settings

1. Install NPM packages and Bower ( this command will install both )

    ```bash
    npm install 
    ```

2. Run Project 
     ```bash
     gulp serve-dev
     ```
3. Have Fun :)
